# 快速排序

数组排序 
- 冒泡 两两相邻位置 比较
- 选择排序 选择最小的元素 放到当前位置
- 插入排序 从当前位置开始 与前面的元素比较 直到找到合适的位置
O(n^2)

## 快排 O(nlogn)
- pivot 基准值 ， 一轮后顺序是确定的
- 比他小的方左边
- 比他大的方右边
双指针交换 

- 递归 对左边和右边的数组分别再做

快排（quick sort） 是一种基于**分治策略**的排序算法， 运行高效（Onlogn）（平均情况）, 应用广泛
分而治之

## 递归，分治的区别
递归是代码实现方式（函数自身调用）
自顶向下 缩小问题的规模
分治是算法思想， 绝大多数用递归实现
分 + 治 + 合 三步 
分 把大问题切成互相独立的小问题
治 解决小问题
合 合并小问题的结果

 左边pivot右边

 相等元素相对位置颠倒，完美体现快排不稳定。

 ## 总结
 - 快的原因 
  pivot 基准值（Ologn） + 原地交换 O(n) 双指针
 - 不稳定的原因
  相等元素相对位置颠倒，完美体现快排不稳定。
- 分治策略
