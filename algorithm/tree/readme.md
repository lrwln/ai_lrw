## 深度与广度优先遍历
- 深度优先：先访问子节点，再访问兄弟节点
- 广度优先：先访问兄弟节点，再访问子节点
  相当于读书 前者正常读书 读章节 然后读章节里的小结 然后继续下个章节 后者先读完目录 在依次读小结


- 深度优先遍历算法---leetcode 104二叉树的最大深度/111 二叉树的最小深度
  - 访问根节点
  - 对根节点的children挨个进行深度优先遍历(这个过程是递归的) 


- 广度优先遍历算法
  - 新建一个队列 将根节点入队
  - 把队头出队并访问
  - 把队头的children挨个入队//二叉树中就是把左子树和右子树入队
  - 重复第二 三步直到队列为空



## 二叉树的先中后序遍历/深度优先遍历(递归版)
- 什么是二叉树
- 树中每个节点最多有两个子节点的树叫二叉树

- 先序遍历 
  - 访问根节点
  - 对根节点的左子树进行先序遍历
  - 对根节点的右子树进行先序遍历
- 中序遍历
  - 对根节点的左子树进行中序遍历
  - 访问根节点
  - 对根节点的右子树进行中序遍历
- 后序遍历
  - 对根节点的左子树进行后序遍历
  - 对根节点的右子树进行后序遍历
  - 访问根节点
