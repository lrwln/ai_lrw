## 数组的创建
- ADT
抽象数据类型
连续的存储空间 + 特定的操作
- push, pop, shift, unshift
  - 修改原数组 以下方法都不是纯函数
  - push 返回新的长度
  - pop 返回被删除的元素(尾部)
  - unshift 在队头添加元素 返回新的长度
  - shift 删除队头元素 返回被删除的元素
- new Array(9)
  指定了长度的空数组 [empty x 9]初始化为空
    数组的此位置没有被占用
    不属于任何类型
    arr[0] undefined
    - (new Array(9)).fill(0)

## 遍历的方法
- for 计数循环
  机器化 命令式
  性能最好
- for...of
  语义很好
- forEach
  缺点是无法实现for循环里的break continue return...
  入栈...开销较大
  优点功能强大 
- map、filter、every、some、reduce
  map返回一个全新的数组 纯函数
  filter 筛选 函数结果为true的元素 返回一个全新的数组 纯函数
  every 检查所有元素是否满足条件 返回布尔值
  some 检查是否有元素满足条件 返回布尔值
  reduce 对数组中的每个元素执行一个reducer函数，每一次执行都会将前一个元素的结果作为参数传递给下一个元素，最终返回一个单一的值
