## BEM 国际命名规范(html)
- .page 页面
  开启一个区块 block,
- 分了上下两部分？
   .page__hd 为头部
   .page__bd 为主体
   hd、bd 是block 下的元素
   Block 是 Element 之间用__ 分隔

## BEM 的优势
- 国际规范 大家都遵守
- 结构清晰 便于后续盒模型的搭建
  - 有一个区块需要表达 即视为block
  - 有几个元素要表达 即视作elements
- 简单易读
- 解决了写页面 类名难命名的问题
  如果没有BEM命名规范 需要写许多单词
