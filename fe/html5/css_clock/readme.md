- html 负责结构

    .clock>.clock-face>(.hand*3) 
    div 可省略
    .clock 类名选择器

- CSS 层叠样式表 
    负责样式
- JS  行为（事件）
- 模块化， 职责分离， 一个文件一个功能 
- css link 在头部引入，更早的和html结合，更快的看到
