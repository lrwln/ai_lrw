## js有多少种数据类型？
- 总共八种
- 根据ECMA262 规定 
 - primitive 原始数据类型有7种
 - 引用数据类型有1种


## 在es6发布前，js只有6种数据类型
也就是下面的五种原始数据类型和一种对象类型object


## 原始数据类型
- Number 数值
- Boolean 布尔值
- String 字符串
- Undefined
- Null
- Symbol 唯一
- BigInt 大数

## 复杂数据类型
- Object 对象
  - 创建时在内存中分配空间 空间内存储的是一个地址 地址/指针 真正指向的数据在堆内存中存储
  - 当调用时本质是通过地址访问堆内存中的数据
