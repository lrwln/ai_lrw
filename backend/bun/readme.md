## typescript 
来自微软 是js的超集 添加了类型约束
js 弱类型 经常会出现类型错误
ts 强类型 可以在编译阶段就发现类型错误 提高代码质量和可维护
- 静态类型编译 ts -> js 文件 检查类型或代码错误
- ts 需要编译成 js 才能运行 在运行时不需要 ts 依赖
- ts已经是ai agent的主流语言了 因为它的类型系统可以帮助我们更好地管理复杂的代码和数据结构

## js的易错性
- 浏览器input输入 默认是string类型 但是我们可能需要number类型 这时候就会出现类型错误
- + 加法和字符串拼接 
- 也不报错出现异常 导致错误难以发现和调试
ts 可以在编译阶段就发现这些类型错误 提高代码质量和可维护性
