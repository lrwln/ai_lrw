## 接口
- interface 是oop(面向对象)中的核心概念
- 封装 继承 多态
- interface用于声明一个对象的约束 这个对象有哪些属性或方法
- 一定要实现这个接口 这个对象必须满足接口的约束
- 面向对象编程 - 面向接口编程 设计模式的基础

## restful
- restful是一种设计风格 不是一种协议
- 一切皆资源
- 定义url的规则 资源的名词 + 资源的操作 http方法
带有一定语义规则 restful 规则

不同的资源 在url上应该有不同的路径
