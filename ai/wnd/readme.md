### Prompting 提示词
直接表达你的需求（模糊的）llm 的产出不确定性太那啥了。

AI 得到**精确**的指令时，可以为我们做很多事情
使用Ai 创作软件吧。

### create an app with AI 
Prompting  清晰的指令 
- 继续Prompt  
    vibe coding 
    **Goal** 
    创建一个网页来帮我写生日贺卡。当我提供一个人的姓名、年龄和爱好时，它应该给我返回一条有趣的消息
    **input**
    **output**

- 加上节日风格的标题和配色，让它更好看一些。 
- 把卡片放在右侧展示，让它看起来像生日贺卡的内页。
    **layout**
- 在页面顶部加一个有趣的标题。另外， 不要替换掉旧卡片，
而是保留他们并叠放在新卡片的下方。

vibe coding 日常！ 当使用AI写代码时，通常从一组
基本指令开始，看看得到了什么？ 然后反复告诉AI我想
怎么改进。

当构建软件时，有一些基本的构建块（building blocks）,
可以包含在提示词中。

## Prompt basic rule 
building blocks 

Goal(目标清晰) + Input(用户输入) + Output(输出)
+ Layout(布局) + Features(特色 灵活)

有了vibe coding 工作的Prompt 规则 ，
训练一下， 任何项目。

### 高级 prompt 
```
创建一个网页来帮我写生日贺卡。
当我输入一个人的姓名、年龄和爱好后，它应该返回一条有趣的消息。
使用喜庆的标题和颜色。
把贺卡显示在右侧，并且让它看起来像生日贺卡的内页。
在页面顶部添加一个有趣的标题。
另外，不要替换旧的贺卡，而是保留他们的显示状态，把他们堆叠在新贺卡的下方
```
