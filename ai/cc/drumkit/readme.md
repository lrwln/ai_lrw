## plan mode 
/plan 
降低了prompt门槛
通过询问一些问题，帮助我们组织Prompt，使我们更方便地描述我们的需求。

## 指令
/ 开头 下个cc的指令
/init 初始化
/plan 开启规划模式

## 接手一个项目 
- 可以看文档 claude.md 文件
- 如果没有可以
  /init 初始化
  cc 是我们的员工
  cc 读取了文件代码，把它的理解写入了claude.md 文件
- chat 模式继续了解项目
