# 入手ai,需要搞懂哪几个关键概念？

## agent(智能体)
- agent是一个能够自主决策和执行任务的智能系统。它可以感知环境、处理信息、做出决策并采取行动。agent通常具有学习能力，可以通过与环境的交互不断改进自己的性能。
现在价值链顶端的agent agent 工程师已经取代传统软件工程师
现在用的很多ai产品 本质上都是agent 只是没有被叫做agent Cursor/Claude codex/Codex/悟空/豆包
不只是回答问题，还可以执行任务，甚至是创造性的任务
- 一个agent有多强？ 取决于 用来什么大脑(llm)，用什么工具，拿到了什么信息

## llm
大模型 是agent的大脑，提供了智能体的认知能力和决策能力。它通过大量的数据训练，能够理解和生成自然语言，进行推理和学习。不同的llm具有不同的能力和适用场景，选择合适的llm对于agent的性能至关重要。


llm只负责**推理**和生成 真正的能力来自工具(tool)和信息(information)

## 工具 tool
llm 只有推理和生成能力，工具(tool)提供了agent执行任务所需的功能和资源。工具可以是各种软件、API、数据库等，帮助agent获取信息、处理数据、执行操作等。选择合适的工具能够增强agent的能力，使其能够更好地完成任务。
